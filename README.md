# pyGetNews

获取新闻信息
=======
默认从以下网站获取新闻：
========

中国新闻网(ZXW)

网易新闻(163)


配置（在GetNews.py中修改）：
==
defaultSiteList = ["ZXW","163"]    #新闻源站点设置，默认都获取

命令执行说明：
==
python GetChinaNews.py -d 2017-03-12 2017-03-14 时间范围

python GetChinaNews.py -a 获取最近30天内的新闻

简单构想：
==
其中在页面展示上打算引入时间戳插件以进行时间段的选取，默认状态为获取当日信息。

对于该部分多采用选择形实现，尽量减少使用者的输入工作量。
